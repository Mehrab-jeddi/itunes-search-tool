# itunes-search-tool
Python-Skript zur Suche von Alben über die iTunes API. Praktische Erfahrung mit Requests und JSON.
Dies ist ein kleines Übungsprojekt in Python.  
Es ruft die iTunes API ab und gibt Informationen zu Künstlern, Alben und Trackanzahl aus.  
Das Projekt dient ausschließlich zum Training im Umgang mit:
- API-Abfragen (Requests)
- JSON-Datenverarbeitung
- Konsolenausgabe
