#itunes-search-tool

Ein kleines *Python-Skript* zur Suche von Alben über die iTunes API.  
Dieses Projekt zeigt praktische Erfahrung im Umgang mit *Requests* und *JSON-Daten* und dient als Übungsprojekt zur Vertiefung der Kenntnisse in API-Integration und Datenverarbeitung.

#Projektbeschreibung
Das Skript ermöglicht es, einen Suchbegriff einzugeben und anschließend die iTunes API abzufragen.  
Die Ergebnisse werden in der Konsole ausgegeben und enthalten Informationen zu:
- Künstlername
- Albumtitel
- Anzahl der enthaltenen Tracks
Damit bietet das Projekt einen einfachen Einstieg in die Arbeit mit externen Schnittstellen und die Verarbeitung von JSON-Daten in Python.

Beispielausgabe:
Please enter a search term : beatles
The search returned 50 results.
